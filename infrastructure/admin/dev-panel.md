## 技术决策记录

### 2025-01-16
- **决策**：采用Spring Boot Admin 3.1.0作为监控框架
- **原因**：与Spring Boot 3.1.0完美兼容，提供开箱即用的监控功能
- **影响**：所有微服务需要添加admin-client依赖

### 2025-01-15
- **决策**：使用内嵌Tomcat提供Web界面
- **原因**：简化部署，降低运维复杂度
- **影响**：需要合理配置内存和线程池
